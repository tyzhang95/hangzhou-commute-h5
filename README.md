# 杭州通勤租房助手 H5

这是一个纯静态 H5 页面，无后端依赖。

## 上传方式

把本目录里的 `index.html` 上传到任意支持 HTTPS 的静态托管服务。

常见选择：

- 腾讯云 COS 静态网站
- 阿里云 OSS 静态网站
- Vercel
- Netlify
- Cloudflare Pages
- 自有服务器或公司静态资源服务

## 公众号使用

上线后拿到公网链接，例如：

```text
https://example.com/hangzhou-commute/
```

再用这个链接生成二维码，放到公众号菜单、文章或海报里。

注意：正式上线时，建议给地图服务 Key 配置域名白名单，并尽量使用后端代理隐藏密钥。
